# 💅 Soraya Arruda

Sistema desenvolvido para gerenciamento de uma manicure, permitindo controlar clientes, agendamentos e serviços de forma simples e organizada.

---

## ✨ Funcionalidades

- Login administrativo
- Cadastro de clientes
- Agendamento de horários
- Controle de serviços
- Dashboard administrativo
- Firebase Authentication
- Firebase Firestore
- Interface responsiva

---

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript
- Firebase
- Firestore
